    Обращение бывают след типов:
            - теги - р
            - класс - .
            - id - #
            - атрибуты - []
            - обращение ко всем элементам - *
    
    Абсолютные пути подключение:
            - Чтобы войти в ту или иную папку необходимо использовать /
            - Чтобы выйти из какой либо папки нужно использовать ..
            - Выйти и зайти будет выглядеть: ../

    Контейнер
        display - это блок отображение (внешнего вида)
            - justify-content - это выравнивание по ширине (по горизонтали)
            - align-items - э то выравнивание по высоте (по вертикали)

        Alt для img всегда надо прописывать.

        border-radius- 
            - радиус округляется до половины (50%) ширины котрые мы задали. 
            Для бордер-радиуса в начале должны всегда задавать ширину и высоту, а потом радиус округлять.
            Ед.изм. указываем тоже что и ширину и высоту.

        > - означает, что обращение будет происходит только для родительских элементов.

    Эффект наведения:
        Задача:
            по наведению на элемент меню отобразить подробные данные.

        Решение:
            1.по наведению = :hover (это ссылки и другие элементы, на которые навели курсор мыши)
            2.на элемент меню = li  (или другой тег, уточняем где именно находится hover меню, внутри класса .hover_m = .hover_m li )
            3. отобразить подробные данные :
                Прежде чем решить, нужно понять следующее:
                    1.как скрыть тот или иной блок?
                        ответ:
                            1.если у него стоить display: none;
                                то решение является его противоположность
                                = display: block (или другие значение кроме none)
                            
                            2.если у него стоить opacity: 0;
                                то решение является его противоположность
                                = opacity: 1;
                Решение равно = display: block 
            4.подробные данные = dropdouwn_m

            Общее решение:

        .hover_m li:hover .dropdown_m {
            display: block;
        }
